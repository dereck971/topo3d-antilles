Mode d'emploi des exports

Chaque format a son usage. Voici ce que vous recevez et comment l'utiliser dans votre logiciel.

PDF

Fiche Parcelle

19€

Fiche A4 synthétique : surface cadastrale, identifiant IDU, code INSEE, altitude moyenne et dénivelé, risques naturels (Géorisques), contour GeoJSON.

Ouvrir avec : tout lecteur PDF (Adobe, Preview, Chrome)

Comment l'utiliser

  1. Ouvrez le PDF reçu par email
  2. Vérifiez les données de base de la parcelle
  3. Joignez la fiche à votre dossier de faisabilité
PDF+

Fiche Light

29€

Tout Parcelle + zonage PLU (U, AU, A, N), profil altimétrique 42 points (transects N-S et E-W avec analyse de pente), carte de ruissellement, servitudes et informations complémentaires.

Ouvrir avec : tout lecteur PDF (Adobe, Preview, Chrome)

Idéal pour

  1. Étude de faisabilité : le PLU vous dit si le terrain est constructible
  2. Le profil altimétrique révèle les pentes et contraintes de terrassement
  3. La carte de ruissellement identifie les zones d'écoulement des eaux
OBJ

Mesh terrain 3D (.OBJ)

Dès 59€ — Essentiel

Maillage 3D triangulé du terrain réel, basé sur les données LiDAR IGN (résolution 2-5m selon l'offre). Coordonnées locales, axe Y = altitude.

Compatible : SketchUp, Blender, Rhino, 3ds Max, Cinema 4D, Unity, Unreal

Import dans SketchUp

  1. Fichier → Importer → sélectionner .obj
  2. Cocher "Conserver l'unité de dessin" (mètres)
  3. Le terrain apparaît en tant que groupe — clic droit → Éclater si besoin
  4. Posez vos volumes de projet directement sur le mesh

Import dans Blender

  1. File → Import → Wavefront (.obj)
  2. Vérifier l'échelle (Forward: -Z, Up: Y)
  3. Le mesh est directement exploitable pour modélisation ou rendu
DXF

Courbes de niveau (.DXF)

Dès 129€ — Complet

Fichier AutoCAD R12 avec les courbes de niveau mineures et majeures, le contour parcelle et les points cotés. 4 calques séparés pour un contrôle total.

Compatible : AutoCAD, ArchiCAD, Revit, QGIS, Vectorworks, BricsCAD

Import dans AutoCAD

  1. Ouvrir directement le fichier .dxf
  2. Les 4 calques sont déjà séparés : CONTOURS_MINEURES, CONTOURS_MAJEURES, PARCELLE, POINTS_COTES
  3. Vérifier l'unité (mètres) dans Propriétés du dessin
  4. Superposer avec votre plan de masse pour caler le projet

Import dans Revit

  1. Insérer → Lier CAO → sélectionner le .dxf
  2. Unités : Mètre, Positionnement : Automatique — Centre à Centre
  3. Les courbes apparaissent dans une vue en plan — utiliser "Surface topographique" pour créer le terrain Revit
GEO

Contour parcelle (.GeoJSON)

Dès 59€ — Essentiel

Contour géoréférencé de la parcelle cadastrale au format GeoJSON standard (WGS84 / EPSG:4326). Utilisable dans tout SIG ou outil cartographique.

Compatible : QGIS, ArcGIS, Google Earth Pro, Mapbox, geojson.io

Visualiser rapidement

  1. Rendez-vous sur geojson.io
  2. Glissez-déposez le fichier .geojson dans la fenêtre
  3. Le contour apparaît sur la carte — vous pouvez l'éditer ou l'exporter en KML/Shapefile
IFC

Format BIM (.IFC)

249€ — Premium

Fichier IFC 2x3 pour intégration directe dans les workflows BIM. Le terrain est modélisé comme un IfcSite avec la géométrie 3D. Interopérable entre tous les logiciels BIM.

Compatible : Revit, ArchiCAD, Allplan, Tekla, BIMcollab

Import BIM

  1. Dans Revit : Ouvrir → IFC → le terrain se crée automatiquement
  2. Dans ArchiCAD : Fichier → Ouvrir fichier → IFC
  3. Le site contient les métadonnées : commune, altitude, surface, coordonnées GPS
STL

Impression 3D (.STL)

249€ — Premium

Mesh fermé et optimisé pour l'impression 3D. Maquette physique de votre terrain avec exagération verticale ajustée pour une bonne lisibilité du relief.

Compatible : Cura, PrusaSlicer, Bambu Studio, Meshmixer, tout slicer FDM/SLA

Impression 3D

  1. Ouvrir le .stl dans votre slicer (Cura, PrusaSlicer...)
  2. Positionner le modèle à plat, vérifier les dimensions
  3. Recommandé : 0.2mm layer height, 15% infill, PLA
  4. La maquette fait ~15×15 cm par défaut — redimensionner selon votre plateau

Questions fréquentes

Tout ce que vous devez savoir sur les exports Topo3D.

Quelle est la précision des données d'altitude ?
Nos fichiers utilisent le RGE ALTI de l'IGN, la meilleure source disponible en France. Résolution native de 1 à 5 mètres selon la zone. En offre Essentiel, la grille est échantillonnée à 5m. En Complet et Premium, elle est à 2m — suffisante pour une étude de faisabilité ou un dépôt de permis. Pour un relevé officiel (bornage, implantation), un professionnel agréé reste nécessaire.
En combien de temps je reçois mes fichiers ?
La génération est automatisée. Vous recevez vos fichiers par email sous 24h maximum après confirmation du paiement. En pratique, c'est souvent quelques heures. Les abonnés Pro bénéficient d'un accès instantané via l'API.
C'est adapté pour quelles phases de projet ?
Topo3D est conçu pour les phases amont : esquisse, faisabilité, études préliminaires, présentation client, demande de permis. C'est l'outil idéal pour avancer rapidement sur un projet sans attendre des semaines. Notre service vous fait gagner 3 à 6 semaines en début de projet.
Les fichiers sont-ils géoréférencés ?
Le GeoJSON est en WGS84 (EPSG:4326). L'OBJ et le DXF utilisent un système de coordonnées locales (centré sur le coin sud-ouest de la parcelle) pour faciliter l'import dans les logiciels de modélisation. Le rapport inclut les coordonnées GPS de référence pour recaler dans un SIG si nécessaire.
Quelles communes sont couvertes ?
Guadeloupe (32 communes, 100%), Martinique (34 communes, 97%), Guyane et La Réunion (disponibles). 66+ communes couvertes en LiDAR HD. La France métropolitaine fonctionne aussi via les APIs IGN.
Et si je ne suis pas satisfait ?
Nous proposons une garantie satisfait ou remboursé. Si les fichiers ne correspondent pas à la parcelle demandée ou présentent un défaut technique, nous régénérons gratuitement ou vous remboursons intégralement.
