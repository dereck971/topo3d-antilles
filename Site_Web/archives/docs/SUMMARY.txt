✅ API ENDPOINTS (9 Serverless Functions in /api/)

1. login.js (2.8 KB)
   - POST /api/login
   - Credential validation (BETA_USER, BETA_PASS)
   - Rate limiting: 5 attempts/minute per IP
   - Session token generation + httpOnly cookie
   - Returns: {ok: true} or {error: "..."}

2. elevation.js (1.9 KB)
   - GET /api/elevation?lat=X&lon=Y
   - Fetches IGN MNT elevation data
   - 10s timeout protection
   - Returns: {elevation, lat, lon, resolution, source}

3. contours.js (2.7 KB)
   - GET /api/contours?lat=X&lon=Y
   - Fetches cadastral parcelles via API Cadastre
   - Returns GeoJSON with parcel properties

4. generate-obj.js (2.3 KB)
   - POST /api/generate-obj
   - Generates Wavefront OBJ 3D mesh
   - Simple triangulation algorithm
   - Returns: OBJ file (text/plain)

5. generate-dxf.js (2.6 KB)
   - POST /api/generate-dxf
   - Generates AutoCAD DXF R12 format
   - Parcel boundary + contour layers
   - Returns: DXF file (text/plain)

6. generate-geojson.js (3.0 KB)
   - POST /api/generate-geojson
   - Generates GeoJSON FeatureCollection
   - Calculates area, bounding box, elevation stats
   - Returns: GeoJSON with metadata

7. fiche-parcelle.js (7.6 KB)
   - POST /api/fiche-parcelle
   - Generates branded PDF regulatory report (15€)
   - Fetches 5 regulatory datasets in parallel
   - PDF sections: Localisation, Élévation, Risques, Zones protégées
   - Returns: PDF file (application/pdf)

8. email.js (1.7 KB)
   - POST /api/email
   - Sends transactional emails via Resend API
   - From: commandes@topo3d-antilles.com
   - Returns: {ok: true, messageId}

9. webhook.js (4.2 KB)
   - POST /api/webhook
   - Stripe webhook handler for checkout.session.completed
   - Verifies signature with STRIPE_WEBHOOK_SECRET
   - Triggers confirmation email
