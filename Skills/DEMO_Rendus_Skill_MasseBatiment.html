Aperçu des rendus possibles

Voici les 4 types de rendus que le skill "masse-batiment-3d" pourrait générer pour les rapports KCI et les livrables clients.

DÉMO TECHNIQUE — Bungalow T2 40m² · Le Gosier · Zone UB
RENDU 1 — RÉALISTE Export: PNG, PDF
Clic + glisser = rotation

Vue perspective architecturale

Rendu semi-réaliste avec matériaux (bois, béton, tôle), ombres portées, végétation tropicale. Idéal pour le rapport KCI Premium 299€ — section "Simulation volumétrique". Le client voit son projet comme s'il était déjà construit.

RENDU 2 — ISOMÉTRIQUE Export: PNG, SVG, PDF
Clic + glisser = rotation

Vue isométrique technique

Style diagramme avec code couleur par fonction (séjour, chambre, SdB). Montrer les reculs PLU en pointillés. Parfait pour l'offre Complète 129€ — lisible, technique, pro. Les archis adorent ce style.

RENDU 3 — PLAN DE MASSE Export: PNG, SVG, DXF, PDF

Plan de masse 2D annoté

Vue de dessus avec cotations, reculs PLU, emprise au sol, zones constructibles. Exportable en DXF pour AutoCAD. C'est le livrable technique de base — celui qui va dans les dossiers permis de construire et que les BET utilisent.

RENDU 4 — TOPO + BÂTIMENT Export: OBJ, glTF, HTML
Clic + glisser = rotation

Intégration terrain topographique

Fusion du MNT (skill topo-3d-parcelle) avec la masse bâtiment. Le client voit son bâtiment posé sur le vrai relief de sa parcelle avec courbes de niveau. C'est le rendu le plus impressionnant — exclusif offre Premium.
